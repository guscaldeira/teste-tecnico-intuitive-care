# Teste Técnico - Engenharia de Dados

Este repositório contém a solução para a **Questão 1** do teste técnico, focada em Web Scraping, ETL e Automação.

##  Questão 1: Monitoramento de Demonstrações Contábeis (ANS)

**Objetivo:** Automatizar a coleta de dados financeiros das operadoras de planos de saúde diretamente do site da ANS, processar arquivos de "Eventos e Sinistros" e consolidar as informações.

###  Como Executar

**Pré-requisitos:**
* Java JDK 11 ou superior.
* Maven.

**Passo a passo:**
1. Clone o repositório.
2. Na raiz do projeto, instale as dependências:
   ```bash
   mvn clean install
